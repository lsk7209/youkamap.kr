## 0. 콘셉트 재정의 — 권위 콘텐츠 중심

검색되는 진짜 수요는 "내 근처 최저가"(→앱)가 아니라 **"왜 이 가격인가 / 언제 내려가나 / 세금이 얼마인가"**다. 이건 앱이 인덱싱 페이지로 못 내놓고, 사람들이 실제로 헷갈려한다.

**3개 트랙으로 재편:**

| 트랙 | 핵심 | 특성 |
|---|---|---|
| A. 권위 에버그린 | 유류세 구조, 가격 결정 원리, 셀프/알뜰/브랜드 차이 해설 | 안정 트래픽, 권위 뿌리 |
| B. 시의성 캡처 | 유류세 인하/환원 이벤트, 국제유가 급변 시 영향 분석 | 스파이크, 고의도, 에피소드성 |
| C. 데이터 보조 | 지역 비교·추세 페이지, 개별 주유소 데이터 | 내부 링크·근거 제공, 대부분 noindex |

→ 기존 "지역 프로그래매틱 대량 발행"은 C로 강등. A·B가 트래픽을 끌고, C가 데이터 신뢰도를 받친다.

---

## 1. 핵심 도구 — 유류세 분해 계산기 (day-1 moat)

```
입력: 주유 금액(또는 리터) + 유종
출력: ┌ 세전 가격
      ├ 교통에너지환경세(정액)
      ├ 교육세 / 주행세
      ├ 부가가치세
      └ 유통마진 추정
      → "당신이 낸 ___원 중 세금은 ___원(__%)"
```

- 시계열 누적 불필요 → **콜드 스타트 무관, 런칭 즉시 가동.**
- 앱 미제공, 정보형 쿼리("기름값 세금 비중")와 직결, task 완료형(SCA-safe).
- **주의**: 세율은 정책 변동(인하/환원)에 따라 바뀜 → 세율을 설정값으로 분리, 발행 전·변동 시 실측 갱신. 현재 인하 시행 여부 단정 금지.

---

## 2. 시의성 트랙 (B) 운영

- 유류세 인하/환원, 국제유가 급변 등 이벤트 발생 시 **"실제 주유소 가격 영향"** 분석 페이지 발행.
- 우리 누적 데이터로 "발표 후 N일간 실제 평균가 변화"를 보여주면 = 언론·앱이 못 주는 사후 검증 콘텐츠.
- 단정 금지 사항: 현재 정책 상태는 발행 시점 실측. 데이터 보고 지연 감안해 "잠정" 표기.

---

## 3. LPG wedge (별도 클러스터)

- LPG차 운전자(택시·특정 차종) 대상, 주유소 대비 경쟁 낮음.
- "LPG 충전소 가격 비교", "LPG차 유지비 vs 휘발유" 가이드 + LPG 데이터(오피넷 LPG 데이터셋).
- 트래픽 규모는 작으나 저경쟁 → 초기 인덱싱 성공 확률 높음.

---

## 4. 법률 강화 (정법무 반영)

- **게이트 0 확장**: (a) 공공누리 유형, (b) **'제공기관 별도 이용조건' 필드** 둘 다 확인. 별도 조건 존재 시 보수적 해석·문의.
- **평가어 금지**: 조건부 섹션·코멘터리에서 "바가지/비싸다" 류 금지. "전국 평균 대비 +X원", "상위 N%" 등 **사실 진술만**.
- **가격 면책 위치**: 가격 표기 인접에 "오피넷 공개 기준, 실제가와 차이 가능, [기준일]" 노출. 푸터 고지로 불충분.
- **출처·재게재**: "출처: 한국석유공사 오피넷, data.go.kr" 강제. 내용 무단 변경 금지(오피넷 정책), 단순 오류정정 외 가공 시 가공 사실 명시.

## 6. 개발 순서 (불변 — 게이트 우선)

```
게이트 0    라이선스 (공공누리 유형 + 별도 이용조건)
게이트 0.5  ① DataLab(세금/인하/셀프 쿼리 수요)  ② 파일 아카이브 백필 가능?
──────── 통과 + 우선순위 재확인 후 ────────
Phase 1  인프라
Phase 2  수집 + 시계열 누적 시작
Phase 3  보강 + 품질/SCA 게이트
Phase 4  사이트: 트랙 A(권위) + 도구 우선 → B(시의성) → C(데이터)
Phase 5  자동화 + 애드센스
Phase 6  런칭
```

## 7. 패널 평결 (5인)

- v0.3에서 콘셉트의 **무게중심이 "지역 데이터"에서 "유가 구조·정책 권위 콘텐츠 + day-1 도구"로 이동.** SCA·콜드 스타트·앱 경쟁을 동시에 우회하는 더 단단한 구조.
- 그러나 GO는 아님: K1(라이선스)·K2(수요)는 그대로. 콘셉트가 강해졌어도 **0 투자 게이트 통과가 선행 조건.**
- **패널 한계효용 경고**: 5인에서 신규 실질(유류세 moat, 법률 정밀화)이 나왔으나, 7인은 마지막 정밀화(수익 정량 모델·법률 자문 확정) 외엔 신규 가치가 낮을 전망. 7인을 돌리기보다 **게이트 0.5(DataLab 실측)로 넘어가 가설을 데이터로 치는 것**을 권고.
