# 유가지도 페르소나 및 콘텐츠 운영 가이드

## 사이트 포지션

유가지도는 한국 운전자가 기름값을 이해하고 비교할 수 있도록 유가 구조, 유류세, 지역별 가격 차이, 절약 전략을 짧고 검증 가능한 문장으로 설명하는 정보 사이트다.

## 핵심 페르소나

1. 실용형 일반 운전자
   - 관심사: 오늘 넣을지, 며칠 기다릴지, 얼마를 아낄 수 있는지.
   - 필요한 답: 리터당 차이, 월 주유량 기준 절약액, 주유 시점.
   - 문체: 숫자를 먼저 보여주고 이유는 뒤에서 설명한다.

2. 정책 이벤트 검색자
   - 관심사: 유류세 인하, 환원, 고시 시행일, 종료일.
   - 필요한 답: 시행일, 대상 유종, 리터당 영향, 실제 가격 반영 시차.
   - 문체: 발표 내용과 추정치를 분리해 표시한다.

3. 지역 비교형 운전자
   - 관심사: 내 지역이 전국 평균보다 비싼지, 옆 지역으로 이동할 가치가 있는지.
   - 필요한 답: 전국 평균 대비 차이, 가까운 대안 지역, 고속도로/도심 차이.
   - 문체: "비싸다" 대신 "전국 평균 대비 +X원"처럼 비교 기준을 쓴다.

4. LPG/알뜰 니치 검색자
   - 관심사: LPG 차량 유지비, 알뜰주유소, 셀프주유소, 고속도로 주유소.
   - 필요한 답: 조건별 장단점, 연간 비용 차이, 예외 상황.
   - 문체: 결론을 단정하기보다 운행거리와 지역 조건을 함께 제시한다.

## 글 작성 규칙

- 외부 글 생성 API를 사용하지 않는다. 초안과 검수는 에디터가 직접 수행한다.
- 최신 수치가 필요한 문단에는 기준일을 명시한다.
- 출처는 본문 가까이에 배치한다. 권장 출처는 오피넷, data.go.kr, 정부 고시다.
- 정책 발표 수치와 편집부 추정치를 같은 표에 섞을 때는 열 이름에 "고시"와 "추정"을 분리한다.
- AI 보조 사용 여부는 글 하단에 고지한다.

## 추천 글 구조

1. 3줄 요약
2. 목차
3. 무엇이 바뀌는지
4. 언제부터 적용되는지
5. 내 주유비에 얼마나 영향을 주는지
6. 지역/유종별 예외
7. 자주 묻는 질문
8. 출처와 업데이트 기준일

## 금지 표현

- 지역이나 주유소를 "바가지", "악덕", "무조건 비쌈"처럼 단정하지 않는다.
- 출처 없는 "곧 오른다", "무조건 싸진다" 같은 예측형 제목을 쓰지 않는다.
- 광고 영역을 실제 콘텐츠처럼 보이게 만들지 않는다.
